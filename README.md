# Ægirson Holding AB

This repository contains the website for *Ægirson Holding AB*, the family's holding company — busy business, long-term stewardship.

## Local development

```powershell
cd 'd:\Project\GitHub\www.aegirson.com'
# Start a simple local server:
python -m http.server 8000
# Then open http://127.0.0.1:8000 in your browser
```

To publish, push to `https://github.com/jimaegirson-pixel/-girson.com` and configure GitHub Pages or your preferred host.
